## 🛠️ The Tech Stack

*   **Framework:** Next.js (App Router)
*   **Styling:** Tailwind CSS & Custom CSS variables
*   **Animations:** Framer Motion (orchestrated springs, entry sequences, and hardware-accelerated transforms)
*   **Scrolling:** Lenis (smooth inertial scrolling, scroll-locking on overlays)
*   **Icons:** Lucide React & custom optimized SVGs
*   **Typography:** Anurati (for display headers), Satoshi & JetBrains Mono (for body/UI)

---

## 🎨 UI/UX Highlights

*   **Cinematic Preloader Curtain:** A clean, horizontal staggered typography reveal (`"Let's"` -> `"begin"` -> `"the"` -> `"show."`) transitioning out with a cinematic `clip-path` wipe reveal.
*   **Flat Local Time UI:** Clean, minimalist local clock time synced with the preloader rise.
*   **Glassmorphic Infinite Marquee:** Seamless, high-framerate horizontal reviewer testimonial marquee that pauses elegantly on hover.
*   **Curtain Shutter Parallax Footer:** A classy 3D reveal footer where the testimonials/contact section slides up like a curtain to uncover the footer underneath it in sync with the scroll progress.
*   **Hardware-Accelerated Fluid Cursor:** A custom magnetic liquid cursor built on Framer Motion's `MotionValues` that scales and deforms dynamically based on movement velocity with zero rendering overhead.
